Antes de continuar construyendo nuevas páginas (como Servicios), debemos establecer una regla global de diseño para todo el sitio.

El **Home ya define la línea gráfica oficial del sitio**.
Todos los nuevos layouts deben reutilizar exactamente ese sistema de diseño.

No crear nuevos estilos visuales si ya existen en el Home.

REGLA PRINCIPAL

El Home funciona ahora como **Design System base del sitio**.

Todos los componentes nuevos deben reutilizar:

* tipografía
* escalas de tamaño
* estilos de botones
* tarjetas
* sombras
* liquid glass
* espaciados
* colores
* microinteracciones

No inventar variantes nuevas si ya existe un componente equivalente.

COMPONENTES QUE YA EXISTEN

Reutilizar estos componentes creados en el Home:

Navbar
PrimaryButton
SecondaryButton
GlassCard
SectionContainer
CaseStudyCard
InsightCard
Footer
CTASection
SectionTitle
SectionSubtitle

Si se necesitan variantes, extender el componente existente en lugar de crear uno nuevo.

ESTILO VISUAL

Mantener exactamente la misma línea gráfica:

* estética sobria
* uso de liquid glass
* gradientes suaves
* imágenes vibrantes
* alto contraste tipográfico
* mucho espacio en blanco

No usar colores planos fuertes.

Usar la paleta de iData ya definida.

LAYOUT

Mantener el mismo sistema de espaciado del Home.

Secciones amplias con buen aire.

Evitar layouts saturados.

RESPONSIVE

Mantener exactamente el mismo comportamiento responsive definido en el Home.

COMPONENTES REUTILIZABLES

Todo elemento que se repita debe ser un componente reutilizable.

Evitar código duplicado.

OBJETIVO

Que todo el sitio mantenga una identidad visual consistente con el Home y funcione como un sistema de diseño coherente.
